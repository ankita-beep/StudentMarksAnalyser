# Student Marks Analyzer

A simple Java console application to analyze student marks using OOP concepts.

## Project Structure
- `Student.java`: Defines student attributes.
- `MarksAnalyzer.java`: Contains logic for calculations.
- `Main.java`: Entry point for user interaction.

## Features
- Input student details (Name, Roll No, Marks).
- Calculate Total, Average, and Grade.
- Find Highest and Lowest marks per student.

## How to Run
1. Compile the code:
   ```bash
   javac *.java
   ```
2. Run the application:
   ```bash
   java Main
   ```
